## 习题
1. 习题一:根据如下代码计算div、ul、和li的最终样式
```
    //html代码
    <div class="list list-member">
        <ul>
            <li>member ID1</li>
            <li>member ID2</li>
            <li>member ID3</li>
        </ul>
    </div>

    //css代码
    .list li {
        margin: 10px 0 0 0;
        font-size: 14px;
        color: #666;
    }

    .list-member li {
        font-size: 12px;
    }
```

提示：需要考虑浏览器默认样式，样式权重，以及样式继承。先思考，给出一个答案，在写一个简单的网页，将代码复制上去到浏览器验证
